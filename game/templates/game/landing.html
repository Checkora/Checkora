About the Project

What is Checkora?

Checkora is an open-source chess platform that fuses a high-performance C++ minimax engine with a Django-powered web frontend. Designed for players who want a real challenge and developers who want to learn engine architecture.

⚙️

Dual-Engine Architecture

At its core, Checkora runs a compiled C++ binary for move generation, validation, and AI search. The engine communicates with Django via subprocess pipes — sending board states as 64-character strings and receiving structured responses. If the C++ binary is unavailable (e.g., on shared hosting), an identical Python fallback engine takes over transparently, so gameplay never breaks.

🧠

Minimax + Alpha-Beta Pruning

The AI uses a minimax search tree with alpha-beta pruning for optimal play. Search depth adapts dynamically — going deeper in endgames (≤6 pieces → depth+2) for stronger tactical play when it matters most.

📖

Opening Book System

In early moves, the AI consults a JSON opening book keyed by FEN strings instead of running expensive search. Multiple book moves per position add variety — no two games feel the same.

⚡

DP Move Caching

Valid moves are computed once per piece position and stored in a dynamic programming cache. The cache auto-invalidates after every move, ensuring 100% accuracy with zero redundant calculation.

🏰

Full Rule Compliance

Every rule is implemented — castling (kingside & queenside), en passant, pawn promotion (Queen/Rook/Bishop/Knight), check, checkmate, stalemate, and three types of draw detection.

📝

PGN & FEN Support

Every game generates Standard Algebraic Notation (SAN) via the C++ engine. Full FEN parsing lets you set up any board position. Games serialize to PGN format for analysis or export.

🌍

Open Source — GSSoC 2026

Checkora is proudly part of GirlScript Summer of Code 2026. Every feature, from the engine to the UI, is open-source and built by a global community of contributors.

Platform Features

Everything You Need to Play

From the chessboard UI to AI difficulty scaling — Checkora is designed to be a complete chess experience right in your browser.

Play vs AI (PvE)

Challenge Checkora's hybrid AI engine. Choose your color (White or Black), and the AI adapts its search depth based on game phase — playing cautiously in openings and aggressively in endgames.

AI Engine

Play vs Friend (PvP)

Play locally with a friend on the same device. The board flips perspective each turn, with independent chess clocks for both players and full move history tracking.

Multiplayer

Chess Clocks & Timers

Each player gets a configurable timer (default: 10 minutes). Clocks tick in real time, pause during game pauses, and trigger automatic timeout detection if a player runs out of time.

Timing

Move History & PGN Export

Every move is recorded in Standard Algebraic Notation (SAN). View the full move list in real-time, and the engine generates PGN-format game records you can analyze externally.

Analysis

Draw Detection Suite

Checkora detects all draw conditions automatically — stalemate, insufficient material, threefold repetition (tracked via position keys), and the 50-move rule (100 half-moves). Players can also offer/accept draws manually.

Rules Engine

Player Statistics

Track your wins, losses, and draws across sessions. View your performance history with detailed game-by-game breakdowns — all stored securely with Django's authentication system.
